Example ValuesList of potential values that fall under each propertyPMEnsures engineers understand your metric; this component is best when done exhaustively and all possible property values are included.

PlatformsValue to indicate which platforms the event triggers fromPMEnables proper coding of the source code, also serves as reminder of platforms to check when testing instrumentation
